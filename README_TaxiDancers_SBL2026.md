## Objet

Page web de planning des **Taxi Dancers** pour les soirées Salsa de l'**Espace Caraïbes** (4 soirées : Mercredi 13 → Samedi 16 mai). Elle présente, pour chaque créneau horaire de chaque soirée :

- Le **lieu** (Dôme ou Playa) et la **transition** Playa → Dôme (Jeudi / Vendredi à 23h)
- L'**animation** du moment (Cha-Cha, Rueda Micro HF, Anim' Ligne, Ban Biyo, Concert…)
- Les **DJs** en poste (avatar rond avec photo)
- Les **Taxi Mambos** et **Taxi Cubanos** de la rotation 30 minutes (avatars ronds)
- Le **bracelet fluo** comme signe visuel de reconnaissance des Taxi Dancers pour les festivaliers

## Bracelet fluo

Le **bracelet fluo** (vert phosphorescent) est le signe de reconnaissance officiel des Taxi Dancers pour les festivaliers. Il est représenté en petit badge lumineux `▬ Fluo` à côté de chaque groupe de taxis sur la page.

Chaque Taxi Dancer porte son bracelet pendant les créneaux de rotation.

## Ajouter / modifier un Taxi Dancer

1. Déposer la photo `.jpg` dans `TAXI CUBANOS/` ou `TAXI MAMBOS/`
2. Nommer le fichier `PRENOM.jpg` (ou `PRENOM NOM.jpg`)
3. Ouvrir `Planning_TaxiDancers_SBL2026.html` et ajouter un bloc `.taxi-chip` dans chaque créneau concerné :
```html
<div class="taxi-chip">
  <div class="taxi-av c"><img src="TAXI CUBANOS/PRENOM.jpg" alt="Prénom"></div>
  <div class="taxi-n">Prénom</div>
</div>
```
4. Remplacer la classe `c` par `m` pour un Taxi Mambo
5. Sauvegarder et rouvrir dans le navigateur
